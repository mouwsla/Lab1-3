# Lab1-3

Набор лабораторных работ на C++ по сортировкам и AVL-деревьям.

## Состав проекта

| Файл | Описание |
| --- | --- |
| `lab1.cpp` | Генерация одномерного массива случайных целых чисел и сортировка по возрастанию. |
| `lab2.cpp` | Генерация одномерного массива случайных целых чисел и сортировка по убыванию методом выбора. |
| `lab3.cpp` | Генерация строк вида `число-число-число` и их лексикографическая сортировка. |
| `Tree-1.cpp` | Реализация AVL-дерева с добавлением ключей и выводом дерева симметричным обходом. |
| `AVLtree-1.cpp` | Меню для работы с AVL-деревом: добавление, удаление, вывод дерева, поиск минимума и максимума. |

## Требования

- C++ compiler with C++11 support or newer
- Terminal for input/output

## Сборка и запуск

Каждый исходный файл является отдельной программой и содержит собственную функцию `main()`, поэтому файлы нужно компилировать по отдельности.

Пример сборки:

```bash
g++ -std=c++11 lab1.cpp -o lab1
g++ -std=c++11 lab2.cpp -o lab2
g++ -std=c++11 lab3.cpp -o lab3
g++ -std=c++11 Tree-1.cpp -o tree
g++ -std=c++11 AVLtree-1.cpp -o avltree
```

Пример запуска:

```bash
./lab1
./lab2
./lab3
./tree
./avltree
```

Для программ сортировки сначала вводится размер массива или количество строк. Для программ с AVL-деревом ввод зависит от подсказок в консоли.
